# 🚀 Material Order Management System

> 🌟 A modern web form to manage and track material orders efficiently
> 📦 Built with simplicity – powered by Google Sheets

---

## ✨ Features

* 📦 **Product Input Form**

  * Enter product name
  * Select product type (Blister, Carton, Medal, Label, Color Box)

* 🧪 **Production Stage Selection**

  * FS / EP / FEP / PP / PS

* ⏱️ **Smart Deadline Calculation**

  * Auto-calculate expected completion date
  * Based on selected range (3–5 days, 5–7 days)

* 📊 **Google Sheets Integration**

  * Data stored instantly
  * Real-time tracking

* 🎨 **Modern UI/UX**

  * Glassmorphism design
  * Smooth animations
  * Responsive layout

## 🚀 How to Use

### 1. Setup Google Apps Script

* Create a Google Sheet
* Open **Extensions → Apps Script**
* Paste the backend code (`doPost`)
* Deploy as Web App:

```
Deploy → New deployment
Type: Web app
Execute as: Me
Who has access: Anyone
```

---

### 2. Connect Frontend

* Replace your Web App URL inside:

```javascript
fetch("YOUR_SCRIPT_URL")
```

---

### 3. Run Project

* Open `index.html`
* Fill form → Submit 🚀
